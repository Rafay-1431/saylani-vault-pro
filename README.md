Saylani Vault - Cloud Learning Portal 🎓

Saylani Vault is a full-stack web application designed to provide students with cloud-based lecture recordings. This project utilizes Firebase Firestore as a backend to ensure that lectures and data are synchronized in real-time.

🔥 Features

Real-time Synchronization: Powered by Firebase Firestore, lectures update instantly across all devices without requiring a page refresh.

Admin Dashboard: A secure administrative panel where authorized users can add, manage, and delete video content.

Cloud Hosting: The project is deployed via Firebase Hosting for global accessibility and high performance.

Responsive UI: A modern, mobile-first design that works perfectly on smartphones, tablets, and desktops.

YouTube Integration: Seamless playback experience using the YouTube IFrame API.

🛠️ Tech Stack

Frontend: HTML5, CSS3 (Custom Grid & Flexbox Layouts), JavaScript (ES6+)

Backend/Database: Google Firebase (Firestore Cloud Database)

Authentication: Firebase Auth (Secure Admin Login System)

Design Assets: FontAwesome Icons & Google Fonts (Montserrat)

🚀 How to Run Locally

Clone the repository:

git clone [https://github.com/YOUR_USERNAME/saylani-vault-pro.git](https://github.com/YOUR_USERNAME/saylani-vault-pro.git)


Configure Firebase:
Open index.html and replace the placeholder firebaseConfig object with your actual Firebase project credentials.

Launch:
Open index.html in any modern web browser or use a "Live Server" extension in your code editor.
